# JavaScript Foundations

A collection of hands-on JavaScript projects built to strengthen core frontend fundamentals through building real features.

## Included Projects

### 1) To-Do App (Vanilla JavaScript)
A browser-based to-do app built with HTML, CSS, and vanilla JavaScript.

**Features**
- Add tasks (button click or Enter key)
- Prevent empty tasks
- Prevent duplicate tasks (case-insensitive)
- Mark tasks complete/incomplete (checkbox + strikethrough)
- Edit tasks (Save/Cancel, Enter to save, Escape to cancel)
- Delete tasks
- Persist tasks using localStorage
- Reorder tasks (↑ / ↓)
- Bulk actions: Clear Completed, Clear All (with confirmation)
- Optional: Filters (All / Active / Completed) + stats

**Tech used**
- HTML / CSS
- Vanilla JavaScript
- Browser APIs: localStorage
- Git + GitHub

## What I’m Learning
- DOM manipulation and event handling
- Application state vs UI state
- Rendering UI from state (render function pattern)
- JSON.stringify / JSON.parse for persistence
- Debugging browser errors with DevTools (Console + Network)
- Git workflow (add → commit → push)

## Running locally
1. Clone this repo
2. Open the folder in VS Code
3. Open `index.html` in a browser or use Live Server
